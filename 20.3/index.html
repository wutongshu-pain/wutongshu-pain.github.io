梧桐树：为了无痛的欢呼！

张学广 2020-03-03 梧桐树

梧桐树下的欢呼，算是应运而生，是偶然堆积的必然结果。任何一种新生，皆来源于前一次的失败。

# NO.1

新冠肺炎疫情，对我的影响就是，呆在家里那也不去，有足够的时间挥霍，干想干的事情，这才能够一个人从零开始，完成了构思、设计、构建框架、写稿、作图、编辑、审校、样本雏形、基本功能菜单等公众号运营的整个流程。

在此要特别感谢小伙伴们组成的三个帮帮团，给予及时的意见反馈和技术改进：新媒体顾问组，大众审核组，专业审核组。

# NO.2

脱下白大褂，写文是我的头号乐趣，医疗圈皆知丁香园，我盯着各个兼职职位半年之久，最后写简历、选职位、过初试、考笔试，然而兼职作者的愿望还是以失败落幕。那干脆就自己试试吧，寄人篱下，始终是别人手中的枪。

失败里找新生，挫折有时很有温度

# NO.3

疼痛医生学习渠道有限，源想法是录制全国各地的会议和培训视频、翻译国外的学习资源，分期推送，调查之后发现，太难了。每个人的学问，不可也不能空手送人，这是必须要维护的良性生态。 靠人不如先靠己，还是要做原创的内容。

# NO.4

病例库是一直都有的思路，只是没有足够的时间细想如何实施，共享视频和资源的否定，间接刺激了构建病例库的想法，趁此机会把概念做成实体。病例库是和原创内容并行的重点。

# NO.5

有一种坚持从未改变：做好传统意义上的医生，是起码的责任和基本形式。但我始终相信，医院之外，还能做很多有价值的事情。

基于以上原因，梧桐树下的欢呼踏着七彩祥云来到大家面前，以下是几点具体设想。

1.倾听疼痛医生心声的 Pain Home ，可以学习的可靠平台（目前能力有限，医护专业方面内容暂列入计划）

从多种渠道网罗疼痛学习资源，介绍临床经验、特殊案例、临床感悟、国际前沿、科研小结等！历经此疫情，所有人都应该明白一个道理，一个国家医生强大，人民健康才有保证。

2.临床医生关于疼痛科普的野外战场

人民健康不能只靠医护救治，医生有责任和义务普及健康知识，让人民群众有更多渠道真实客观的了解疾病，认识疾病，做好提前预防，尽早治愈。这不仅减轻患者的痛苦和负担，更减轻了医护的工作压力。

3.疼痛患者了解疼痛的便捷途径，人民大众在疼痛方面的健康热线

  • 疼痛到底是什么
  • 疼痛科都治什么病
  • 我的疼痛属于什么病
  • 有什么有效的治疗方法
  • 到底能不能痊愈

疼痛患者和健康人群对于疼痛疾病的疑惑，在这里有必要做好解答。

4.建设医患共同参与的疼痛病例库

同一种疾病的患者在病例库进行交流，分享病情，以沟通缓解患病焦虑，以交流加深病情理解，让病友之间的共情不再遥远。

病人之间不清楚的共性问题，有专业医生热心解答，让医患沟通更进一步。在这里，医疗质量和人文关怀延续到了医院之外。

每天分散的接诊不同病人，注定漏掉容易忽视的细节，如果每天花一点点时间与很多同一类型的病人进行交流，再小的细节也会放大，这是一个提高自己的便捷途径。做一个合格的医生不难，做一个好医生太难了，区别无非是后者关注细节，有着对疾病的全面把握。

病人是医生最好的老师，医生需要了解患者真实感受是什么？担忧是什么？面对的问题是什么？想做一个好医生，医患接触不仅仅是门诊的五分钟和病房的住院期间。

知识的更大价值在于分享、传播和应用，希望更多医生参与进来，每个人都可以成为优质内容的输出者和分享者，抱团取暖，越走越远。

以上皆是纸上谈兵，关于公众号真正的意义，并不在这里，它真实的蕴含在每一篇推送的内容里。
